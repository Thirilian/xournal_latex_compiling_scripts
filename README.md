# xournal_latex_compiling_scripts
Small scripts that can be used seemlessly by Xournal app to make latex writing faster, easier and more efficient.

I have been noticing that xournal didn't allow to generate a formula if its code wasn't compilable. It makes sense but at the same time, I wanted to be able to close the latex coding  window to carry on with note taking and go back to my latex code later if the formula was incorrect.
So I asked for help on the xournal github () () and as it turns out, I got a tipp.
